Class1.cs - файл библиотеки классов, из которого берётся весь функционал сервиса: перемещение, шифрование(используется класс Aes), архивация, деархивация, дешифрование файла. Также там находятся классы-парсеры, в зависимости от расшиерния файла специальный метод выбирает, какой парсер использовать. 

Service1.cs - файл windows-службы, где происходит вызов класса, содежращего все конфигурации. Также в этом файле располагается метод, который записывает событие создания файла или папки в отслеживаемоей директории в отдельный текстовый файл.

mws.json - файл конфигурации json.

config.xml - файл конфигурации xml.
При запуске "основного" файла Service1.cs создаётся файл службы MyWindowsService.exe. Переменная options запускает дженерик метод получения данных из файла конфигурации, а затем эти данные используются для корректной работы класса, в котором вызываются все нужные методы службы.
